# excel-business-finance-pm-portfolio 
Excel portfolio showcasing finance, data analysis, dashboards, and project management.

---

## Repository Structure

| Folder | File | Purpose |
|--------|------|---------|
| `01_formulas` | `sample_data.xlsx` | Base sales-style data and basic calculations |
| `02_formatting_charts` | `sales_formatting.xlsx` | Conditional formatting, tables, and basic charts |
| `03_pivot_analysis` | `sales_data_analysis.xlsx` | Sales analysis using summaries and pivot-style analysis |
| `04_dashboard` | `sales_chartsanddashboard.xlsx` | Sales charts and dashboard for visual analysis |
| `05_financial_statements` | `financial_statement.xlsx` | Practice financial statements (P&L, Balance Sheet) |
| `06_project_management` | `project_management_tracker.xlsx` | Project task tracker |
